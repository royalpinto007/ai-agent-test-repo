# SDLC Agent

An AI-powered software development pipeline built on GitHub Issues, Claude Code CLI, and n8n. Open an issue, assign it to `agent-accellier`, and a chain of AI agents handles the full lifecycle — from requirements analysis to deployment — with human approval at every stage.
